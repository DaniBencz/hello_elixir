__to run `.ex` files__:
`elixir <filename>.ex`

__to run `.exs` files__:
- `iex "<filename>.exs"` or `iex -r *.exs`
- call the functions as `<Module>.<Function>` within the terminal

__Notes__:
- after changing script code, run `r(<Module>)` in `iex` to update it in memory
